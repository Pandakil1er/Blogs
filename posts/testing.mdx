## 🛠️ The Setup Process

<Steps>
  <Step title="Initialize the project">
    Start by cloning the repository and installing dependencies. 
    Make sure you have Node.js 18+ installed.
  </Step>

  <Step title="Configure MDX Components">
    Import your custom components into the `mdx-components.tsx` file to make them globally available or import them locally in your `.mdx` files.
  </Step>

  <Step title="Deploy to Vercel">
    Push your code to GitHub and connect it to Vercel for automatic deployments.
    <Callout type="success" emoji="✅">
      Vercel will automatically detect the Next.js framework and optimize your build.
    </Callout>
  </Step>
</Steps>
